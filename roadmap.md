## Phase 2: Core Algorithms (Weeks 4-7)
*Target: 25-30 problems | ~22 hours total*

### Week 4: Stacks & Queues
- Monotonic stack patterns
- Queue implementations
- **Key problems:** Valid Parentheses, Min Stack, Daily Temperatures, Sliding Window Maximum

### Week 5: Binary Search
- Standard binary search
- Search space problems
- Rotated arrays
- **Key problems:** Search in Rotated Array, Find Minimum, Koko Eating Bananas, Median of Two Sorted Arrays

### Week 6: Trees - Part 1
- Tree traversals (DFS: preorder, inorder, postorder)
- BFS level order
- **Key problems:** Invert Tree, Max Depth, Same Tree, Level Order Traversal

### Week 7: Trees - Part 2
- BST operations
- Path problems
- **Key problems:** Validate BST, Kth Smallest, Lowest Common Ancestor, Binary Tree Max Path Sum

## Phase 3: Advanced Patterns (Weeks 8-11)
*Target: 25-30 problems | ~22 hours total*

### Week 8: Graphs - Part 1
- BFS/DFS on graphs
- Connected components
- **Key problems:** Number of Islands, Clone Graph, Pacific Atlantic Water Flow

### Week 9: Graphs - Part 2
- Topological sort
- Shortest paths (Dijkstra, Bellman-Ford basics)
- **Key problems:** Course Schedule I & II, Network Delay Time

### Week 10: Dynamic Programming - Part 1
- 1D DP patterns
- Climbing stairs, house robber patterns
- **Key problems:** Climbing Stairs, House Robber, Coin Change, Longest Increasing Subsequence

### Week 11: Dynamic Programming - Part 2
- 2D DP
- String DP
- **Key problems:** Unique Paths, Edit Distance, Longest Common Subsequence

## Phase 4: Expert Topics (Weeks 12-14)
