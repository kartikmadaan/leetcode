## 💡 Study Tips

1. **Time yourself** - Practice with 45-minute limits for coding, 45 minutes for system design
2. **Explain out loud** - Verbalize your thought process
3. **Write clean code** - Use meaningful variable names, add comments
4. **Review patterns** - Focus on recognizing problem patterns, not memorizing solutions
5. **Spaced repetition** - Revisit problems you struggled with after a few days
